- **지인 소개** — 주변에 관심 있을 만한 분께 자연스럽게 알려주기 (가장 쉬움)

### 발송 조건 메모

- **조회 쿼리**: `partners` 테이블에서 `created_at < now() - interval '3 days'` AND `partner_programs` 테이블에 해당 partner_id 레코드 0건
- **프로그램 리스트 동적 삽입**: 현재 `is_active=true`인 모든 프로그램을 테이블 형식으로 삽입
- **중복 발송 방지**: 한 번 발송된 파트너에게 재발송하지 않음 (발송 로그 테이블 또는 `email_sent_flags` 컬럼 필요)
- **구현 위치**: `/api/cron/nudge-inactive-partners` (신규 Cron 엔드포인트)
- **Vercel Cron**: `vercel.json`에 매일 09:00 KST 실행 스케줄 등록

**활동 가이드: 이렇게 시작하세요**

**STEP 1. 가장 쉬운 방법부터**

| 난이도 | 채널 | 방법 |
|--------|------|------|
| 쉬움 | **지인 소개** | 주변에 관심 있을 만한 분께 카톡으로 추천 링크 전달 |
| 쉬움 | **카톡/카페** | 관련 커뮤니티에 정보 공유 형식 글 + 추천 링크 |
| 보통 | **블로그** | 검색 키워드 중심 후기/정보글 + 하단에 추천 링크 |
| 보통 | **인스타그램** | 릴스/피드 콘텐츠 + 프로필 링크에 추천 링크 등록 |
| 어려움 | **유튜브** | 쇼츠/롱폼 영상 + 더보기란에 추천 링크 |

**STEP 2. 콘텐츠 작성 팁**

- 직접 경험하거나 써본 느낌을 솔직하게 전달하세요
- "광고"보다 "정보 공유" 톤이 반응이 좋습니다
- 추천 링크는 글 하단 또는 프로필에 자연스럽게 배치하세요
- 한 채널에서 꾸준히 하는 게 여러 채널에 흩뿌리는 것보다 효과적입니다

**STEP 3. 실적 확인**

대시보드에서 실시간으로 내 추천 현황을 확인할 수 있습니다:
- 추천 링크 클릭 수
- 문의 접수 건수
- 유효/계약 확정 건수
- 누적 커미션

**자주 묻는 질문**

**Q. 추천 링크와 추천 코드의 차이는?**
추천 링크로 유입되면 자동 추적됩니다. 추천 코드는 링크를 쓸 수 없는 상황(전화, 오프라인 등)에서 고객이 직접 입력하는 용도입니다.

**Q. 커미션은 언제 받나요?**
문의 → 유효 확정 → 계약 완료 → 정산 순서로 진행됩니다. 각 단계마다 이메일로 알려드립니다.

**Q. 여러 프로그램에 동시 참여할 수 있나요?**
네, 마켓플레이스에서 원하는 프로그램을 추가로 신청하실 수 있습니다.
