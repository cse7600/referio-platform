# 밀리의서재 - 에어브릿지 포스트백 설정 가이드

> Referio 연동을 위한 에어브릿지 설정 안내서입니다.
> 아래 2단계를 순서대로 진행해주세요.

---

## 배경

에어브릿지에서 Referio로 전환 데이터를 보내려면 **2단계 설정**이 필요합니다:

1. **조직(Organization) 레벨**에서 엔드포인트(수신 URL)를 먼저 등록
2. **앱(밀리의서재) 레벨**에서 해당 엔드포인트로 데이터를 보내는 커넥션을 추가

---

## 1단계: 조직 엔드포인트 설정

에어브릿지에 "데이터를 보낼 주소"를 등록하는 단계입니다.

### 진입 방법 (두 가지 중 하나)

- **방법 A**: 에어브릿지 대시보드 **우측 상단 프로필 아이콘** 클릭 > **조직 설정** 선택
- **방법 B**: 대시보드 상단에 표시되는 **"엔드포인트 설정하기"** 버튼 클릭 (버튼이 보이는 경우)

### 설정 순서

1. 좌측 메뉴에서 **연동 관리(Integration)** 클릭
2. **엔드포인트(Endpoint)** 탭 선택
3. **+ 엔드포인트 추가** 버튼 클릭
4. 아래 정보를 입력합니다:

| 항목 | 입력값 |
|------|--------|
| **이름(Name)** | `Referio Webhook` (자유롭게 변경 가능) |
| **URL** | `https://referio.puzl.co.kr/api/webhook/airbridge` |
| **Method** | `POST` |

5. **헤더(Header) 추가** 버튼을 클릭하고 아래 값을 입력합니다:

| 헤더 키(Key) | 헤더 값(Value) |
|-------------|---------------|
| `X-API-Key` | `94818dfd1485459fbe124dbae88315f3` |

6. **저장** 버튼 클릭

> 이 단계가 완료되면 "Referio Webhook"이라는 엔드포인트가 목록에 나타납니다.

---

## 2단계: 광고주 포스트백 커넥션 추가

1단계에서 등록한 엔드포인트로 실제 이벤트 데이터를 보내도록 설정하는 단계입니다.

### 설정 순서

1. 에어브릿지 대시보드에서 **밀리의서재 앱**을 선택합니다
2. 좌측 메뉴에서 **연동(Integration)** > **광고주 포스트백(Advertiser Postback)** 클릭
3. **+ 커넥션 추가(Add Connection)** 버튼 클릭
4. **엔드포인트 선택**: 1단계에서 만든 **"Referio Webhook"** 선택
5. **전송할 이벤트 선택**:
   - **가입(Sign Up)** 이벤트: 체크
   - **구독(Subscribe)** 이벤트: 체크
   - 그 외 필요한 이벤트가 있으면 추가 체크
6. **전송할 속성(Properties) 설정** (매우 중요):
   - 속성 목록에서 **`sub_id`** 항목을 반드시 포함시켜 주세요
   - `sub_id`는 어떤 파트너가 유입시킨 사용자인지 식별하는 핵심 값입니다
   - 체크가 안 되어 있으면 Referio에서 파트너를 매칭할 수 없습니다
7. **저장** 버튼 클릭

---

## 설정 완료 후 확인 요청

설정이 완료되면, 아래 내용을 Referio 팀에게 전달해 주세요:

### 필수 전달 사항

1. **Preview Payload 캡처**
   - 커넥션 설정 화면에서 **"Preview payload"** 버튼을 클릭하면
     실제로 전송될 JSON 데이터 구조를 미리 볼 수 있습니다
   - 이 화면을 **스크린샷으로 캡처**하여 Referio 팀에게 전달해 주세요
   - Referio 측에서 데이터 구조에 맞게 수신 처리를 최종 보정합니다

2. **이벤트 이름 확인**
   - 가입 이벤트의 정확한 이름 (예: `sign_up`, `airbridge.user.signup` 등)
   - 구독 이벤트의 정확한 이름 (예: `subscribe`, `airbridge.user.subscribe` 등)

### 선택 확인 사항

3. **방화벽 설정** (해당되는 경우)
   - 에어브릿지에서 데이터를 보내는 서버 IP가 방화벽에 의해 차단될 수 있습니다
   - 에어브릿지 측에 발신 IP 목록을 문의하여, 필요 시 allowlist에 추가해 주세요
   - Referio 서버(`referio.puzl.co.kr`)는 별도 IP 차단 없이 수신 가능합니다
