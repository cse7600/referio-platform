## 1. 개요

### 1-1. 목적

Referio(어필리에이트 마케팅 SaaS)에 유입된 광고주에게 퍼즐코퍼레이션의 다른 마케팅 서비스(퍼포먼스 마케팅, 바이럴/UGC 마케팅, 콘텐츠 제작)를 자연스럽게 소개하여 크로스 세일즈를 유도한다.

### 1-2. 비즈니스 배경

- Referio는 퍼즐코퍼레이션이 운영하는 어필리에이트 플랫폼
- Referio 광고주 = 이미 "파트너 추천을 통한 고객 확보"에 관심 있는 마케팅 니즈가 있는 기업
- 파트너 채널(Earned Media)만으로는 성장 한계가 있음 → Paid + Owned 채널 조합이 필요
- 퍼즐코퍼레이션은 퍼포먼스/바이럴/콘텐츠 마케팅 풀퍼널 서비스를 제공하므로 시너지가 명확
- 광고주가 Referio 사용 중 자연스럽게 퍼즐코퍼레이션 서비스를 인지하고 문의로 이어지게 함

### 1-3. 크로스 세일즈 로직

| Referio에서의 상황 | 니즈 연결 | 제안 서비스 |
|-------------------|----------|------------|
| 추천 리드는 들어오는데 전환율이 낮음 | 구매 CAC 최적화, 전환율 개선 | 퍼포먼스 마케팅 (Growth Marketing) |
| 파트너 채널 외 추가 인지도 확장 필요 | 인플루언서·체험단·UGC로 브랜드 검색량 확대 | 바이럴/UGC 마케팅 |
| 파트너에게 줄 소재·콘텐츠 부족 | 숏폼·크리에이티브·블로그 콘텐츠 제작 | 콘텐츠 제작 (Creative) |

## 3. 기능 명세

### 3-1. 노출 위치: 광고주 사이드바 신규 메뉴

**경로**: `/advertiser/grow`
**메뉴 명칭**: "마케팅 성장" (사이드바 아이콘: TrendingUp 또는 Sparkles)
**메뉴 위치**: 현재 사이드바에서 "프로그램" 아래, "설정" 위

> 별도 페이지로 구성하는 이유: 충분한 정보(서비스 설명, 성과 사례, 신뢰 지표)를 담으려면 독립 페이지가 필요. Settings 하단 섹션은 콘텐츠가 묻힐 위험이 있음.

### 3-2. 페이지 구성 (`/advertiser/grow`)

#### 상단 헤더 섹션

```
제목: "파트너 마케팅 그 이상을 준비하세요"
부제: "Referio로 파트너 채널을 구축했다면, 이제 퍼포먼스·바이럴·콘텐츠로 풀퍼널 성장을 시작하세요."
```

#### 섹션 A: 서비스 카드 (3개)

각 카드에 포함할 내용:

**카드 1 — 퍼포먼스 마케팅 (Growth Marketing)**
- 아이콘: BarChart2
- 태그: 구글 프리미어파트너 · 메타 비즈니스파트너 · 토스애즈 공식대행사
- 제목: "데이터 기반으로 CAC를 낮추세요"
- 설명: 네이버·구글·메타·토스 등 주요 매체를 통합 운영하여 구매 전환율과 광고 효율을 극대화합니다. 파트너 리드와 유료 광고 리드를 함께 관리하면 전체 전환율이 향상됩니다.
- CTA 버튼: "퍼포먼스 마케팅 문의"

**카드 2 — 바이럴/UGC 마케팅**
- 아이콘: Users
- 태그: 인플루언서 · 체험단 · 블로그/인스타/유튜브 통합
- 제목: "브랜드 검색량을 500% 높이세요"
- 설명: 인플루언서·체험단이 직접 생성한 콘텐츠(UGC)로 브랜드 인지도를 빠르게 확산합니다. 파트너 추천과 UGC가 결합되면 신뢰도와 전환율이 함께 올라갑니다.
- CTA 버튼: "바이럴 마케팅 문의"

**카드 3 — 콘텐츠 제작 (Creative)**
- 아이콘: FileVideo
- 태그: 숏폼 · 크리에이티브 소재 · SEO 블로그
- 제목: "파트너가 바로 쓸 수 있는 소재를 만드세요"
- 설명: 숏폼 영상, 배너, 온사이트 블로그까지 목적에 맞는 콘텐츠를 제작합니다. 파트너에게 제공할 소재가 풍부할수록 추천 전환율이 높아집니다.
- CTA 버튼: "콘텐츠 제작 문의"

#### 섹션 B: 신뢰 지표 배너

```
수치/아이콘 3개 배치:
- 구글 프리미어파트너 | 메타 비즈니스파트너 | 토스애즈 공식대행사
```

**사례 1**
- 클라이언트: 세이브택스

> "퍼포먼스 마케팅과 바이럴 마케팅이 유기적으로 결합되면서 빠르게 매출 성장을 경험했습니다." — 에스엔디인터내셔널

```
버튼: "무료 상담 신청"  →  puzl.co.kr 문의 페이지 또는 mailto:contact@puzl.co.kr
```

### 3-3. CTA 동작 방식

**방식 A (권장)**: 클릭 시 새 탭으로 `https://puzl.co.kr` 열기 (단순, 빠른 구현)
**방식 B**: 클릭 시 인터널 문의 폼 모달 (이름, 회사, 연락처, 관심 서비스 체크박스) → `contact@puzl.co.kr`로 이메일 발송

> 초기 구현은 방식 A. 방식 B는 추후 전환율 추적이 필요할 때 고도화.

- **외부 API 호출 없음**: 모든 콘텐츠(서비스 설명, 사례, 신뢰 지표)는 정적 데이터로 하드코딩

## 5. UI/UX 가이드라인

### 톤앤매너

- 광고처럼 느껴지지 않게 → "추가 구매 유도"가 아니라 "성장 파트너십 제안" 톤 유지
- 구체적인 숫자(ROAS 1000%, 검색량 500%)로 신뢰 기반
- "Referio로 파트너 채널을 시작했다면, 다음 단계는 풀퍼널" 이라는 자연스러운 흐름
- 퍼즐코퍼레이션 고유 슬로건 활용: "성과중심 풀퍼널 마케팅"

### 디자인 참고

- 서비스 카드: 기존 광고주 대시보드의 카드 스타일 유지 (흰색 배경, border, rounded-xl, shadow-sm)
- 배지/태그: `bg-blue-50 text-blue-700 text-xs` 스타일
- CTA 버튼: `variant="default"` (primary 버튼)
- 하단 CTA 배너: `bg-gray-900 text-white` 다크 섹션

### 레이아웃

- 서비스 카드 3개: `grid-cols-1 md:grid-cols-3` 반응형 그리드
- 성공 사례 2개: `grid-cols-1 md:grid-cols-2`
- 신뢰 지표: flex justify-center gap-8
